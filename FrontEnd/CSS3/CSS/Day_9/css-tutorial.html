01 CSS Selectors
Universal Selector — *

Targets every single element on the page at once. Most commonly used to reset default browser spacing so you have a clean starting point.

Example
* {
  margin: 0;
  padding: 0;
}
Element Selector

Selects all elements with the same HTML tag name. Every matching tag on the page receives the declared styles.

Example
p {
  color: blue;
}
Class Selector — .name

Targets elements sharing a class attribute. Multiple elements can carry the same class name.

Example
.card {
  background-color: lightblue;
}
ID Selector — #name

Targets a single unique element using its ID. An ID must appear only once per page.

Example
#header {
  background-color: black;
}
Descendant Selector

Selects elements nested anywhere inside another element using a space between selectors.

Example
.container p {
  color: red;
}
Child Selector — >

Targets only direct children of a parent. Deeper nested elements are not affected.

Example
.container > p {
  color: green;
}
02 CSS Specificity
What is Specificity?

Specificity is the rule browsers use to decide which CSS style wins when multiple rules target the same element. The selector with the highest specificity always takes priority.

Priority Order — Highest to Lowest
1,0,0,0
Inline Style — highest priority
0,1,0,0
ID Selector — #id
0,0,1,0
Class Selector — .class
0,0,0,1
Element Selector — tag
Specificity Example

When the same element is targeted by an ID, a class, and a tag selector — the ID selector wins and red color is applied.

Example
#title { color: red; } /* wins */
.heading { color: blue; }
h1 { color: green; }
03 Typography
font-family

Defines the typeface used for displaying text. Always include a generic fallback like sans-serif.

Example
p {
  font-family: Arial, sans-serif;
}
font-size

Controls the size of text. Accepts px, rem, em, or % units.

Example
h1 {
  font-size: 32px;
}
font-weight

Controls text thickness. Values: normal, bold, or numbers like 400, 600, 700.

Example
p {
  font-weight: bold;
}
text-align

Aligns text horizontally. Values: left, right, center, justify.

Example
h1 {
  text-align: center;
}
text-transform

Changes capitalisation. Values: uppercase, lowercase, capitalize.

Example
h1 {
  text-transform: uppercase;
}
line-height

Controls vertical spacing between lines. A value of 1.6–1.8 improves readability.

Example
p {
  line-height: 1.8;
}
04 Size Properties
width

Defines the horizontal size of an element from left to right.

Example
div { width: 300px; }
height

Defines the vertical size of an element from top to bottom.

Example
div { height: 200px; }
min-width

Sets the smallest width the element is allowed to shrink to.

Example
div { min-width: 250px; }
max-width

Sets the largest width the element is allowed to grow to.

Example
div { max-width: 600px; }
min-height

Prevents an element from shrinking below a minimum height.

Example
div { min-height: 200px; }
max-height

Restricts an element from growing beyond a maximum height.

Example
div { max-height: 500px; }

What is the Box Model?

Every HTML element is treated as a rectangular box. It has four layers — Content (the actual data), Padding (inner space), Border (the outline), and Margin (outer space from other elements).

content area

Holds the actual text or image. Width and height apply directly to this area.

Example
div {
  width: 300px;
  height: 200px;
}
box-sizing

border-box includes padding and border inside the declared width — prevents unexpected overflow.

Example
* {
  box-sizing: border-box;
}
06 Padding and Margin
padding

Adds inner space between the content and the border. Increases breathing room inside an element.

Example
.card {
  padding: 20px;
}
margin

Adds outer space outside the border. Creates separation between elements on the page.

Example
.card {
  margin: 20px;
}
padding directions

Apply padding individually to top, right, bottom, and left for precise control.

Example
.card {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 10px;
  padding-left: 20px;
}
margin directions

Apply margin individually to each side for precise spacing between specific elements.

Example
.card {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 10px;
  margin-left: 20px;
}
07 Border Properties
border

Shorthand that sets width, style, and color of a border in one line.

Example
.card {
  border: 2px solid black;
}
border-width

Controls the thickness of the border line surrounding an element.

Example
.card {
  border-width: 5px;
}
border-style

Sets the appearance of the border. Values: solid, dashed, dotted, double, none.

Example
.card {
  border-style: dashed;
}
border-radius

Rounds the corners of an element. Used in cards, buttons, and modern UI designs.

Example
.card {
  border-radius: 10px;
}
08 Display Property
display: block

Takes up the full available width and always starts on a new line. Default for div, p, h1–h6.

Example
div { display: block; }
display: inline

Takes only as much width as content needs. Does not start on a new line. Default for span, a.

Example
span { display: inline; }
display: inline-block

Stays on same line but still allows width and height to be set.

Example
button { display: inline-block; }
display: none

Completely hides the element and removes it from layout. It takes no space on the page.

Example
.menu { display: none; }
display: flex

Activates Flexbox — a one-dimensional layout system for rows or columns.

Example
.container { display: flex; }
display: grid

Activates CSS Grid — a two-dimensional layout system for rows and columns.

Example
.container { display: grid; }

display: flex

Turns the container into a flex container. All direct children become flex items arranged in a row by default.

Example
.container { display: flex; }
flex-direction

Sets direction items flow. row = horizontal (default), column = vertical.

Example
.container {
  flex-direction: row;
}
justify-content

Aligns items along the main axis. Values: flex-start, flex-end, center, space-between, space-around.

Example
.container {
  justify-content: center;
}
align-items

Aligns items along the cross axis. Commonly used for vertical centering.

Example
.container {
  align-items: center;
}
flex-wrap

Allows items to wrap onto multiple lines when there is not enough horizontal space.

Example
.container {
  flex-wrap: wrap;
}
gap

Creates uniform spacing between flex items. Cleaner than using margins on each item.

Example
.container {
  gap: 20px;
}

display: grid

Activates CSS Grid — a two-dimensional layout system controlling rows and columns simultaneously.

Example
.container { display: grid; }
grid-template-columns

Defines how many columns the grid has and their widths. The fr unit distributes space proportionally.

Example
.container {
  grid-template-columns: 1fr 1fr 1fr;
}
grid-template-rows

Defines the number and height of rows inside the grid container.

Example
.container {
  grid-template-rows: 100px 100px;
}
gap

Creates gutters between all grid rows and columns for a clean organised layout.

Example
.container {
  gap: 20px;
}

12 Background Properties
background-color

Fills the background with a solid colour. Accepts colour names, hex, rgb, hsl values.

Example
div {
  background-color: lightblue;
}
background-image

Places an image behind the content using the url() function with the image path.

Example
div {
  background-image: url("image.jpg");
}
background-repeat

Controls whether the background image tiles. Use no-repeat to prevent tiling.

Example
div {
  background-repeat: no-repeat;
}
background-size

Controls the size of the background image. cover fills the entire container maintaining ratio.

Example
div {
  background-size: cover;
}
background-position

Sets where the background image is placed. Values: center, top, bottom, left, right.

Example
div {
  background-position: center;
}
13 Position Properties
position: relative

Moves the element from its original place but still keeps that space reserved in the layout.

Example
.box {
  position: relative;
  top: 20px;
}
position: absolute

Removes the element from the normal flow. Positions relative to the nearest positioned ancestor.

Example
.box {
  position: absolute;
  top: 50px;
  left: 50px;
}
position: fixed

Locks the element to the screen. Stays in the same position even when the user scrolls.

Example
.button {
  position: fixed;
  bottom: 20px;
  right: 20px;
}
position: sticky

Acts like relative until a scroll point is reached, then sticks like fixed. Perfect for navbars.

Example
.navbar {
  position: sticky;
  top: 0;
}
14 Pseudo Classes

↑ Live demo — hover and click to see :hover and :active in action

:hover

Applies styles when the mouse pointer is placed over an element. Used on buttons, links, and cards.

Example
button:hover {
  background-color: black;
  color: white;
}
:active

Applies styles at the moment an element is being clicked. Creates a pressed button effect.

Example
button:active {
  transform: scale(0.95);
}
:focus

Applies styles when an input or button receives keyboard or click focus.

Example
input:focus {
  border: 2px solid blue;
}
:first-child & :last-child

Target the first or last child inside a parent. Great for removing borders on edge items.

Example
li:first-child { color: red; }
li:last-child { color: green; }

transition

16 Media Queries
What are Media Queries?

Media queries let you write different CSS rules that activate only when certain screen conditions are met — such as screen width being under 768px. They are the foundation of every responsive website.

Example
@media(max-width: 768px) {
  /* styles applied on screens smaller than 768px */
}
Responsive Text

Reduce font sizes on smaller screens to keep headings from overflowing on mobile devices.

Example
@media(max-width: 768px) {
  h1 {
    font-size: 24px;
  }
}
Responsive Layout

Switch a flex row layout to column on small screens so content stacks vertically instead of overflowing.

Example
@media(max-width: 768px) {
  .container {
    flex-direction: column;
  }
}
