Document Structure Tags

<!DOCTYPE html>

The <!DOCTYPE html> declaration defines the document type and tells the browser that the webpage is written in HTML5. It helps the browser display the webpage correctly using modern HTML standards. This declaration must always appear at the top of the HTML document.

Example:

<!DOCTYPE html>

<html>

The <html> tag is the root element of an HTML document. All other HTML elements are written inside this tag. It represents the beginning and ending of the webpage structure.

Example:

<html>
</html>

<head>

The <head> tag contains information about the webpage such as title, meta tags, favicon links, stylesheets, and scripts. The content inside the head tag is not directly visible on the webpage.

Example:

<head>
</head>

<title>

The <title> tag defines the title of the webpage displayed on the browser tab. It is written inside the head section. The title is also important for search engine optimization.

Example:

<title>HTML Learning Page</title>

<body>

The <body> tag contains all the visible content of the webpage such as headings, paragraphs, images, tables, forms, videos, and links. Everything displayed in the browser window is written inside the body tag.

Example:

<body>
</body>

Meta Tags

<meta charset="UTF-8">

The meta charset tag defines the character encoding of the webpage. UTF-8 supports almost all international languages and special symbols. It helps the browser correctly display text content without encoding errors.

Example:

            <meta charset="UTF-8">

<meta name="viewport">

The viewport meta tag makes the webpage responsive on different screen sizes such as mobile phones, tablets, and desktops. It helps control the page dimensions and scaling behavior.

Example:

            <meta name="viewport"
            content="width=device-width, initial-scale=1.0">

<meta name="description">

The description meta tag provides a short summary about the webpage. Search engines may display this description in search results. It helps improve search engine optimization.

Example:

            <meta name="description"
            content="HTML5 Learning Webpage">

<meta name="keywords">

The keywords meta tag specifies important keywords related to the webpage content. It was mainly used by search engines to identify webpage topics.

Example:

            <meta name="keywords"
            content="HTML, CSS, JavaScript">

<meta name="author">

The author meta tag specifies the name of the webpage author or creator. It provides ownership information about the webpage.

Example:

            <meta name="author"
            content="Faitha Begam">

<link rel="icon">

The favicon link tag is used to display a small icon on the browser tab. It helps identify the website visually when multiple tabs are opened.

Example:

            <link rel="icon" href="logo.png">

Text Tags

<h1> to <h6>

Used to create headings from largest to smallest.

<p>

Used to create paragraphs in webpages.

<br>

Moves content to the next line.

<hr>

Creates a horizontal line between sections.

<pre>

Displays text with original spacing and line breaks.

<span>

Inline container used for styling small text portions.

<div>

Block-level container used to group elements.

<b>

Displays text in bold format.

<strong>

Defines important bold text.

<i>

Displays text in italic style.

<em>

Defines emphasized text.

<u>

Underlines text content.

<mark>

Highlights text with background color.

<small>

Displays smaller-sized text.

<del>

Represents deleted text.

<ins>

Represents inserted text.

<sup>

Displays superscript text.

<sub>

Displays subscript text.

<abbr>

Defines abbreviated text.

<code>

Displays programming code.

<kbd>

Represents keyboard input.

<blockquote>

Defines a long quotation section.

<q>

Defines short inline quotations.

<cite>

Represents reference or source title.

List Tags

<ul>

Creates an unordered list with bullet points.

<ol>

Creates an ordered list with numbers or letters.

<li>

Defines list items inside ul or ol tags.

<dl>

Creates a description list.

<dt>

Defines the title in a description list.

<dd>

Defines description data for dt tag.

Link and Navigation Tags

<a>

Creates hyperlinks to webpages, files, emails, or sections.

<nav>

Defines navigation links section in webpage.

Image and Media Tags

<img>

Displays images in webpages.

<figure>

Groups media content with caption.

<figcaption>

Defines caption for figure element.

<audio>

Embeds audio files into webpage.

<video>

Embeds video files into webpage.

<source>

Specifies media source for audio or video.

<track>

Adds subtitles or captions to media.

<iframe>

Embeds another webpage inside webpage.

<embed>

Embeds external applications or media.

<object>

Embeds multimedia objects in webpage.

<marquee>

Creates scrolling text or images.

Table Tags

<table>

Creates a table to organize data into rows and columns.

<tr>

Defines a row inside a table.

<th>

Defines heading cells in a table. Text appears bold and centered by default.

<td>

Defines normal data cells inside table rows.

<caption>

Provides a title or caption for the table.

<thead>

Groups table header content.

<tbody>

Groups the main table body content.

<tfoot>

Groups footer content in a table.

<colgroup>

Groups multiple columns for styling purposes.

<col>

Defines column properties inside colgroup.

Form Tags

<form>

Creates a form to collect user input such as login details, feedback, registration data, and more.

<label>

Defines labels for form elements.

<input>

Creates input fields for user data. Different input types provide different functionalities.

<input type="text">

Creates a single-line text input field.

<input type="password">

Creates a hidden password input field.

<input type="email">

Accepts only valid email addresses.

<input type="number">

Accepts numeric values only.

<input type="date">

Allows users to select dates.

<input type="time">

Allows users to select time.

<input type="radio">

Creates single-choice selection options.

<input type="checkbox">

Creates multiple-choice selection options.

<input type="file">

Allows users to upload files.

<input type="color">

Opens a color picker.

<input type="range">

Creates a slider input control.

<input type="submit">

Submits form data.

<textarea>

Creates a multi-line text input area.

<select>

Creates a dropdown list.

<option>

Defines options inside select tag.

<optgroup>

Groups related options inside dropdown lists.

<button>

Creates clickable buttons.

<fieldset>

Groups related form elements together.

<legend>

Provides title for fieldset section.

<datalist>

Provides predefined suggestions for input fields while still allowing custom values.

<output>

Displays calculation or result output.

Semantic Tags

<header>

Defines the top section of a webpage or section.

<nav>

Defines navigation links section.

<main>

Represents the main important content of the webpage. Search engines use this tag to identify primary content.

<section>

Groups related content into sections.

<article>

Defines independent content such as blogs, news articles, or posts.

<aside>

Represents sidebar or extra related content.

<footer>

Defines the bottom section of webpage or section.

<details>

Creates expandable and collapsible content.

<summary>

Defines visible heading for details tag.

<time>

Represents date and time information.

<address>

Defines contact information of author or organization.

<mark>

Highlights important text content.

<details>

Creates expandable and collapsible content sections.

<summary>

<dialog>
