Types of Tags

Closing tags

These tags have both opening and closing tags.

Example: p tags, h1 tags, div tags, span tags, a tags, ul tags, li tags etc.

Self closing tags

These tags are closed within the opening tag itself.

Example: br tags, hr tags, img tags, input tags, link tags, meta tags etc.

Empty tags

These tags do not have closing tags and cannot contain any content.

Example: area tags, base tags, col tags, embed tags, source tags, track tags, wbr tags etc.
